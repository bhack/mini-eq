# Security Policy

## Supported Versions

Mini EQ is pre-1.0. Security fixes are provided for the latest public release
and the current `main` branch.

## Reporting A Vulnerability

Please do not report security vulnerabilities in public issues.

Use GitHub's private vulnerability reporting from the repository Security tab.
If private vulnerability reporting is unavailable, open a minimal issue asking
for a security contact and do not include vulnerability details.

Please include the affected version or commit, the platform, reproduction steps,
and the potential impact when you submit a report privately.
